# Stash Plugins

This repository publishes a Stash plugins source index for the plugins stored in [plugins](/plugins).

## Install

In Stash, go to `Settings` -> `Plugins` -> `Available Plugins` -> `Add source`.

- `Name`: `xypat`
- `Source URL`: `https://xypat.github.io/stash-plugins/main/index.yml`
- `Local path`: `xypat`
